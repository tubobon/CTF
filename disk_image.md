# disk解析のメモ
## この手の問題は大体`mount`が必要
mountをするためにはディスクイメージのパーティションを知る必要がある
パーティションが判明したら、`mkdir 任意のパス`で適当にマウント先を作り、`sudo mount -o loop,offset= Start * セクタ数（基本は512バイト）　イメージファイル　パス`でマウント可能
※`mount`してflagを見つけた後に、`unmount`しないと後々めんどいことになるので注意

## その他のイメージファイルに関する便利コマンド

- `mmls`コマンド：diskのパーティションを確認するコマンド
- `fls`コマンド：diskの中にあるファイル一覧を確認するコマンド（削除されたファイルも含めて）
- `testdisk`コマンド：パーティションやファイルを復元するコマンド?
